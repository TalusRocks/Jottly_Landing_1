How it Works

Jott down your ideas directly into Jott.ly’s rich text editor, which auto-saves your work every 20 seconds, keeping it safe.

Store your brilliant ideas in an intuitive and searchable folder system, and share it with your colleagues, permitting them to either edit directly, comment in the margins, or both.

There’s no need to save multiple file versions because everyone is working on the same document! Plus, you can easily revert back to an older version if you need to.
